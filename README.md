## Description

Au3AnswerByTag is a support tool for the Servicedesk employees of [University of Twente](https://www.utwente.nl/en/) in the Netherlands. Answers of repeating questions can be stored within the tool. They can quick and easily found by tags and copied in rich text format style to email program of choice.

#### *Installation*

**Variant 1 - Run the EXE**

Starting the EXE `build\Au3AnswerByTag.exe` which is compiled with `#AutoIt3Wrapper_UseX64=y` and `#AutoIt3Wrapper_UseUpx=n` (the default expected system is x64).

**Variant 2 - Compile it on your own**

Open file `src\Au3AnswerByTag.au3`, adjust the *#AutoIt3Wrapper compiler information* if you want to. For example you could change `#AutoIt3Wrapper_UseUpx=n` from no (n) to yes (y), etc.
Then, run or compile the program.
